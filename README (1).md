# Meal Planner

A smart meal planning web application that helps users plan their weekly meals and automatically generates shopping lists.

## Project Overview

Meal Planner is a web-based tool designed to simplify weekly meal planning and grocery shopping. Users can browse recipes, select meals they want to cook, and instantly receive a comprehensive shopping list with all necessary ingredients. 

## Features

- **Easy Meal Selection**: Browse and select from 12+ popular recipes
- **Smart Shopping Lists**: Automatically combines ingredients and calculates quantities
- **Time Saving**: Plan your entire week in minutes
- **Waste Reduction**: Buy only what you need
- **Print Functionality**: Print your shopping list to take to the store
- **Clean Modern Design**: Professional gradient UI with responsive layout

## Technologies Used

- **Backend**: Python 3 with Flask framework
- **Frontend**: HTML5, CSS3, JavaScript
- **Design**: Custom CSS with gradient backgrounds
- **Font**: Google Fonts (Inter)

## How to Run

### Prerequisites
- Python 3.7 or higher
- pip (Python package installer)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/MelikeBaki/meal-planner. git
cd meal-planner
```

2. Install Flask:
```bash
pip install flask
```

3. Run the application:
```bash
python app.py
```

4. Open your browser at:  `http://localhost:5000`

## How It Works

1. **Browse Recipes**: View available recipes on the home page
2. **Select Your Meals**: Choose dishes you want to cook for the week
3. **Generate Shopping List**: Get a complete list of all ingredients
4. **Go Shopping**: Print or use your phone to check off items
